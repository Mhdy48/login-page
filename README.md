# 🔐 Login Page UI

A sleek, modern, and fully responsive login page with smooth CSS animations and a dark theme. Perfect for your next web project! 🚀

## 📸 Preview

![Login Page Preview](login%20picture.PNG)

> 💡 Open index.html in your browser to see the live preview.

## ✨ Features

- 🎨 Modern dark theme design
- ⚡ Smooth hover & focus effects
- 📱 Fully responsive (mobile, tablet, desktop)
- 🖼️ Rounded avatar with neon shadow
- 💫 Floating label animation when typing
- 🔘 Login button with color transition
- 🔗 Forgot password link

📏 Screen Width 📦 Container Width
Less than 700px 60%

1000px - 1300px 30%
More than 1300px 20%

🎨 Customization

Want to change the accent color? Find #dfe470 in style.css and replace it with your favorite color:

css
/_ Example: change to blue _/
div.login_page input:focus {
border-bottom-color: #4a9eff;
}
div.login_page button:hover {
background-color: #4a9eff;
}
